This is a collection of git cheats and tips I’ve gathered. I’ll add more as time goes by

installing git
  • Install git on macOS with Homebrew
    $ brew install git
  • Install git on Debian-based linux
    $ sudo apt-get install git
  • Install git on Windows with Chocolatey
    $ choco install git
